# Fabric-Warehouse-Optimization-System
# Fabric Warehouse Optimization & Data Analytics System 🏭📊

Bu depo (repository), bir tekstil fabrikasının akıllı depo yönetimi süreçlerini uçtan uca optimize etmek amacıyla geliştirilmiş 3 temel mühendislik ve veri bilimi modülünü içermektedir.

## 📂 Proje Yapısı ve Modüller

Proje, birbirini tamamlayan 3 bağımsız alt çalışmadan oluşmaktadır. Detaylı teknik açıklamalar ve kodlar ilgili klasörlerin altındaki README dosyalarında yer almaktadır:

### 1. [Data Analysis Study in a Fabric Warehouse](./Data-analysis-study-in-a-fabric-warehouse-main)
* **Açıklama:** Kumaş deposundaki mevcut operasyonel verilerin analiz edilmesi, talep tahminleri ve süreç iyileştirme çıktılarını içerir.

### 2. [Application of Genetic Algorithm in Fabric Warehouse Layout](./Application-of-Genetic-Algorithm-in-Fabric-Wa...)
* **Açıklama:** Tesis yerleşimi optimizasyonudur. Depo içi yerleşim düzenini, malzeme akışını maksimuma çıkaracak ve taşıma maliyetlerini minimuma indirecek şekilde **Genetik Algoritma (GA)** kullanarak tasarlar.

### 3. [Application of Route Optimization in a Fabric Warehouse](./Application-of-Route-Optimization-in-a-Fabric...)
* **Açıklama:** Sipariş toplama rotalarının optimizasyonudur. Yerleşimi optimize edilen depo içerisinde en verimli rotayı çizmek için **Gezgin Satıcı Problemi (TSP)** ve **Dijkstra** algoritmalarını kullanır.
